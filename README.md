## Context

Datafluence is a simple API where you can ask for all the public data on an influencer

## Setup

- ### run in dev mode

  - `pnpm i`
  - `pnpm dev`

- ### build

  - `pnpm clean`
  - `pnpm build`

- ### lint

  - `pnpm run lint`

- ### test / coverage

  - `pnpm test`
  - `pnpm test:coverage`
